# telus-wizard

[![version][npm-image]][npm-url] [![Build Status][circle-image]][circle-url]

> Wizard the renders a React component based on the steps you pass to it

## Install

```bash
npm install @telus/telus-wizard
```

For local development, also run:
```bash
npm run setup-local
```
